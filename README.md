# 📚 ST Lorebook Folders & Binder

Удобный менеджер лорбуков для SillyTavern. Группируйте лорбуки по папкам и привязывайте их к конкретным персонажам или отдельным чатам!

## ✨ Главные функции

* 📁 **Система папок:** Создавайте собственные папки и организуйте лорбуки так, как удобно вам. Больше никакой каши в меню!
* 🔗 **Точечная привязка:** Привязывайте лорбук к конкретному персонажу или даже к отдельному чату с ним (идеально для разных веток сюжета или альтернативных вселенных).
* ⚡ **Умная авто-активация:** При входе в привязанный чат, нужные лорбуки автоматически получают статус "активные миры для всех чатов". При выходе из чата — они автоматически отключаются.

## 🛠️ Установка

1. Откройте SillyTavern.
2. Перейдите во вкладку **Extensions** (Расширения) -> **Install Extension**.
3. Вставьте ссылку на этот репозиторий:
https://github.com/Gromova-Olga/ST-Lorebook-Folders
4. Нажмите **Save & Connect** (или перезагрузите сервер).

## 📖 Как использовать

1. На верхней панели SillyTavern появится новая иконка папки — нажмите на неё.
2. Нажмите `+ Папка`, чтобы создать новую категорию.
3. Разверните меню **Привязки к чатам** под нужным лорбуком.
4. Выберите бота и конкретный чат для привязки. Готово!
